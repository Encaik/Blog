# 命令行初始化项目

react-native init [项目名称]

注意

1.项目名称不支持短横线-，应使用下划线或驼峰命名法 2.项目默认使用yarn，创建后勿混用npm 3.项目路径不能包含中文 4.项目启动后无需再安装依赖包,初始化已安装
