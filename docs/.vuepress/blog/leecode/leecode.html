# 1480.一维数组的动态和

难度：简单

题目链接

初次完成代码：

/**
 * @param {number[]} nums
 * @return {number[]}
 */
var runningSum = function(nums) {
  return nums.map(function(t, i) {
    let sum = 0;
    for (let index = 0; index < i + 1; index++) {
      sum += nums[index];
    }
    return sum;
  });
};

# 剑指 Offer 58 - II. 左旋转字符串

难度：简单

题目链接

初次完成代码：

/**
 * @param {string} s
 * @param {number} n
 * @return {string}
 */
var reverseLeftWords = function(s, n) {
  let before = s.slice(0, n);
  let after = s.slice(n);
  return after + before;
};
