# 使用 babel-parser 转化源代码为 AST 示例

被转换的源代码文件 parcode.js:

var a = 1;
var b = 2;

function add(a, b) {
  return a + b;
}

var sum = add(a, b);

console.log(sum);

转化源代码至 AST 的程序 genast.js:

var { parse } = require("@babel/parser");
var fs = require("fs");

fs.readFile("./parcode.js", "utf-8", (err, data) => {
  if (err) {
    throw err;
  }
  var res = parse(data);
  fs.writeFile("./ast.json", JSON.stringify(res), "utf-8", function(err) {
    if (err) {
      throw err;
    }
  });
});

# 使用 babel-generator 转化 AST 为源代码示例

解析 AST 生成源代码的程序 parast.js:

var generate = require("@babel/generator").default;
var fs = require("fs");

fs.readFile("./ast.json", "utf-8", (err, data) => {
  if (err) {
    throw err;
  }
  var res = generate(JSON.parse(data)).code;
  fs.writeFile("./gencode.js", res, "utf-8", function(err) {
    if (err) {
      throw err;
    }
  });
});

生成后的源代码文件 gencode.js:

var a = 1;
var b = 2;

function add(a, b) {
  return a + b;
}

var sum = add(a, b);
console.log(sum);

此处 console 前应有一个空行，暂时不知空行消失的原因

# estree 规范解析

以下是 estree 在 es5 中的节点类型：

  • Node objects(节点对象)
  • Identifier(识别)
  • Literal(文字)
    • RegExpLiteral(正则表达式)
  • Programs(程序)
  • Functions(函数)
  • Statements(声明)
    • ExpressionStatement(表达声明)
    • BlockStatement(块)
    • EmptyStatement(Empty)
    • DebuggerStatement(Debug)
    • WithStatement(与声明)
    • Control flow(控制流)
      • ReturnStatement(return)
      • LabeledStatement(标签声明)
      • BreakStatement(break)
      • ContinueStatement(continue)
    • Choice(判断)
      • IfStatement(if)
      • SwitchStatement(switch)
        • SwitchCase(case)
    • Exceptions(异常)
      • ThrowStatement(throw)
      • TryStatement(try)
        • CatchClause(catch)
    • Loops(循环)
      • WhileStatement(while)
      • DoWhileStatement(dowhile)
      • ForStatement(for)
      • ForInStatement(forin)
  • Declarations(声明)
    • FunctionDeclaration(方法声明)
    • VariableDeclaration(变量声明)
      • VariableDeclarator(变量声明器)
  • Expressions(表达式)
    • ThisExpression(this)
    • ArrayExpression(Array)
    • ObjectExpression(Object)
      • Property(属性)
    • FunctionExpression(方法表达式)
    • Unary operations(一元运算)
      • UnaryExpression(一元表达式)
        • UnaryOperator(一元运算符)
      • UpdateExpression(更新表达式)
        • UpdateOperator(更新运算符)
    • Binary operations(二元运算)
      • BinaryExpression(二元表达式)
        • BinaryOperator(二元运算符)
      • AssignmentExpression(分配表达式)
        • AssignmentOperator(分配运算符)
      • LogicalExpression(逻辑表达式)
        • LogicalOperator(逻辑运算符)
      • MemberExpression(成员表达式)
    • ConditionalExpression(条件表达式)
    • CallExpression(call)
    • NewExpression(new)
    • SequenceExpression(序列表达式)
  • Patterns(模式)

节点在 AST 中的表示接口为:

interface Node {
  type: string;
  loc: SourceLocation | null;
}
