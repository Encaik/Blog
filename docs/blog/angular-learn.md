# 学习Angular9

## 搭建开发环境

- node.js

最新版node.js即可

- Angular cli工具

```sh
npm install -g @angular/cli
```

## 创建新项目

```sh
ng new [project_name]
```

然后根据提示，配置项目。

- 是否需要添加路由？
- 选择要使用的CSS规范
